# Contributing to Sovereign Engine

Thank you for considering contributing to Sovereign Engine! 🎉

## Code of Conduct

Please follow our [Code of Conduct](CODE_OF_CONDUCT.md) to ensure a welcoming environment.

## How to Contribute

### 1. Reporting Bugs

- Open an issue using the **Bug Report** template.
- Include:
  - Steps to reproduce
  - Expected vs Actual behavior
  - PDF example (if possible)
  - Error logs

### 2. Suggesting Features

- Open an issue with the **Feature Request** template.
- Explain the problem and why the feature is needed.

### 3. Submitting Pull Requests

1. Fork the repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Test your code thoroughly
5. Commit your changes (`git commit -m 'Add amazing feature'`)
6. Push to the branch (`git push origin feature/amazing-feature`)
7. Open a Pull Request
